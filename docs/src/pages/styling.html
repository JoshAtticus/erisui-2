Overview

ErisUI styling is handled via standard CSS and extensive use of CSS Variables (Custom Properties). This allows for easy theming and dynamic updates at runtime.

Key Files

styles.css

Contains the core system variables (colors, spacing, typography) and the global reset. Do not modify this heavily if you want to stay consistent with updates.

public/css/custom.css

Intended for your custom styles and overrides. The predefined classes like flex-card are just starter helpers; feel free to replace or extend them.

Theme Variables

Commonly used variables you might want to override in :root:

Variable Description
--app-text Main text color.
--app-accent Primary accent color for interactive elements.
--app-100 to --app-900 Background grayscale ramp.
--nav-width Width of the collapsed navigation.
