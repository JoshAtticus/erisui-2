Overview

The ErisUI router is a custom-built solution designed to handle client-side navigation without the overhead of complex libraries. It uses the History API to manage URL changes and dynamically render content into the main view.

Features

Zero Dependencies

Written in pure vanilla JavaScript, ensuring maximum performance and no bloat.

Hash Support

Handles deep linking and scrolling to anchors even after a page navigation.

Parameters

Supports dynamic route parameters (e.g., /user/:id) for flexible routing.

Transitions

Built-in support for simple page transitions during navigation.

Usage

Routes are defined in src/main.js using the router.add method.

import { router, loadPage } from "./scripts/router.js"; // Define a route router.add("/", () => loadPage("index")); router.add("/about", () => loadPage("about")); // Handle 404s router.setNotFound(() => loadPage("404")); // Initialize window.addEventListener("DOMContentLoaded", () => { router.navigate(window.location.pathname, false); }); API
Method Description
router.add(path, renderFn) Registers a route and its callback function.
router.navigate(path) Navigates to the specified path programmatically.
router.setNotFound(fn) Sets the handler for unmatched routes.
