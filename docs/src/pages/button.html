Usage

Buttons have label text that describes the action that will occur if a user taps a button.

<eui-button>Default</eui-button> <eui-button type="filled">Filled</eui-button> <eui-button type="outlined">Outlined</eui-button> Icon

An icon may optionally be added to a button to help communicate the button's action and help draw attention.

Send Open
<eui-button><eui-icon name="send" width="18" height="18"></eui-icon>Send</eui-button> <eui-button type="filled">Open<eui-icon name="replyIn" width="18" height="18"></eui-icon></eui-button> Link

Buttons can behave as links when the href attribute is present.

Link Button Filled Link
<eui-button href="#">Link Button</eui-button> <eui-button href="#" type="filled">Filled Link</eui-button> Attributes
Attribute Description
type The style type: filled, outlined, or default.
width Fixed width in pixels.
height Fixed height in pixels.
border-radius Border radius in pixels.
icon Sets the button to be a circular icon button.
href The URL that the button points to. Transforms the button into an anchor tag.
