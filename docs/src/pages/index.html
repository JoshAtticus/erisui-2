ErisUI is a library of web components that make it easy to build modern web applications.

Get Started

Components

Avatar

Avatars represent people, groups, or organizations.

Checkboxes

Checkboxes allow users to select one or more items from a set.

Switch

Switches allow users to toggle between two options.

Progress Bars

Progress bars inform users about the status of ongoing processes, such as loading an app or submitting a form.

Text Fields

Text fields allow users to enter and edit text.

Buttons

Buttons prompt most actions in a UI.

Icons

Icons represent concepts, objects, or actions.

Chips

Chips are compact, interactive elements that allow users to select or filter options.
