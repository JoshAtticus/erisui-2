Overview

The eui-app-nav component provides a responsive, collapsible side navigation drawer. It handles routing and active state management automatically.

Usage

Place the eui-app-nav component at the root of your application, typically alongside your main content wrapper.

<eui-app-nav></eui-app-nav>

Unlike other components, eui-app-nav automatically populates itself based on the routing configuration or internal logic (in this demo app). in a real scenario, you might modify `src/components/eui-app-nav.js` to define your links.

Customization

You can customize the width of the navigation drawer using CSS variables.

--nav-width Default: 75px
--nav-expanded-width Default: 250px
Navigation Items

The nav uses eui-nav-item for individual links.

<eui-nav-item path="/" icon="home" label="Home"></eui-nav-item>
Attribute Description
path The URL path to navigate to.
icon The name of the icon to display.
label The text label for the item.
