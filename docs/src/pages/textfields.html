Usage

Text fields behave like input elements. They provide a container with labels for user input.

<eui-input label="Name" id="name" value=""></eui-input> <eui-input label="Password" id="pass" value="" filled type="password"></eui-input> Attributes
Attribute Description
label The label text.
value The input value.
id The input ID.
type The input type (e.g., text, password).
filled Whether the input uses the filled style.
